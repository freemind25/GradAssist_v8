# **App Name**: GradeAssist

## Core Features:

- Interactive Grade Table: Create a grid-based input table with columns for each grade level (A+, A, A-, etc.) and rows for each evaluation criterion (Presentation orale, Presentation Affichée, etc.).
- Automated Calculation: Dynamically calculate the total points for each student based on the selected grade levels for each criterion.
- Export Functionality: Download or export the grading table into a PDF or CSV file.

## Style Guidelines:

- Primary color: Light blue (#B0E2FF) for a professional, clean look.
- Secondary colors: Light grey (#E0E0E0) for table borders and subtle accents.
- Accent: Green (#4CAF50) for highlighting passing grades.
- Clean, grid-based layout to match the paper form for a familiar and efficient workflow.
